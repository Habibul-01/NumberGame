Welcom to Number Game..!

ℹ️INSTRUCTIONS:
  1. It is a game about guessing number generated by the system.
  2. Range of numbers will show on your screen.
  3. The secret number is randomly chosen between "0 and (range - 1)(Which will be generated by the system)".
  4. There are 10 attempts to reach the "Secret" number.
  5. If you guess lower of the actual number it shows "Your Guess is going low".
  6. If you guess higher of the actual number it shows "Your Guess is going High".
